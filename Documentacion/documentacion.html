🌠 ¿Qué soy?

Soy un sistema de íconos basado puramente en HTML, CSS y Javascript. Cuento con la funcionalidad poder darle estilos a cada uno de mis íconos mediante atributos, al igual que cambiar el contenido de un ícono por otro, más adelante veremos en detalle cada uno de los atributos disponibles para dichas acciones. Por último, la mayoría de mis íconos son adaptables a una gran cantidad de tamaños, pudiendo llegar como mínimo a los 25px y como máximo a los 250px (o más).

💫 Dandole estilos a mis iconos

Inicialmente todos los íconos vienen con fondo negro y su contenido es de color blanco, pero podemos cambiarle los estilos haciendo uso de los siguientes atributos, e incluso con uno de ellos podemos cambiar un ícono por otro:

  • f => Establece el color de fondo.
  • c => Establece el color para el contenido del ícono.
  • s => Establece el tamaño del ícono.
  • b => Establece el border-radius.
  • t => Este atributo permite establecer un color de fondo para los íconos que no pueden ser totalmente transparentes. Por favor diríjase a la sección de Advertencias para saber más al respecto.
  • m => Establece el margin.
  • p => Establece el padding.
  • o => Establece la opacidad total del ícono.
  • change-iconejz => Permite cambiar el contenido de un ícono por otro.

Si gustan cambiarle los estilos predeterminados a todos los iconos de un solo golpe, podrían dirigirse al archivo "iconejz.css" y entre las primeras líneas, encontrarán las variables de dichos estilos por lo que pueden cambiar sus valores asignándole los que ustedes deseen que tengan.

Un ejemplo podría ser que los iconos tuviesen inicialmente un color de contenido verde en vez de blanco. Para lograr esto sería tan sencillo cómo modificar la variable "--color-iconejz" dándole el valor de "#29FFAA" se vería de la siguiente forma:

👀 Advertencias

        
        
Hay ciertos íconos que no pueden ser completamente transparentes, ya que se dejan de ver cómo esperamos, la razón de esto es que algunos íconos como "jz-wng" tiene elementos dentro que dan la impresión de ser huecos y visualizar el fondo del ícono, entonces si a estos le establecemos un f="transparent", notaremos qué ya el contenido de adentro, dejará de verse.
