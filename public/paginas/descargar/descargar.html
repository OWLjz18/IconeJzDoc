¡Bienvenidos al apartado de descargas!

Actualmente pueden descargarme de diferentes maneras, usando Git, Github CLI y npm.

Al descargarme, este será el contenido de mi carpeta:

  • src/iconejz.css => Es el archivo CSS dónde se encuentra la magia con la que se dibuja cada ícono :D
  • src/iconejz.js => Es el archivo Javascript que se encarga de toda la parte estructural y funcional (crea los componentes).
  • src/iconejz.txt => Es un archivo de texto plano con el que podrán tener a la mano una lista completa de todos los iconos disponibles.
  • LICENSE.md => Es un archivo Markdown que contiene la licencia del proyecto.
  • README.md => Es un archivo Markdown que sirve para la presentación en Github.
  • package.json => Es un archivo JSON que contiene datos sobre mí para poder estar en npm.

Para que el repositorio sea clonado con exito deben tener la opción de su elección instalado en su sistema operativo, de no tenerlo por favor diríjanse a su sitio web oficial e instalenlo. Una vez instalado pueden proseguir con el proceso a continuación.

Diríjanse a su terminal, en ella navegen hasta la carpeta de su proyecto y ejecuten el comando del programa que hayan escogido usar:

Git

$ git clone 'https://github.com/OWLjz18/IconeJz.git'

En caso de que deseen agregarme como un submódulo a su proyecto: (mi recomendación)

$ git submodule add 'https://github.com/OWLjz18/IconeJz.git'

Github CLi

$ gh repo clone 'OWLjz18/IconeJz'

NPM

$ npm install iconejz
